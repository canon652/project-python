# Сервис для блюра конфиденциальных данных

Проект представляет собой веб-сервис, который автоматически находит и замазывает (блюрит) конфиденциальные данные на загруженных изображениях (JPEG, PNG). Сервис использует OCR (Tesseract) для распознавания текста и нейросеть Gemma для определения ФИО.

## Доступные для блюра типы данных

При загрузке изображения можно выбрать, какие типы данных нужно заблюрить:

- ФИО — полные имена определяются с помощью нейросети Gemma
- Номер телефона — российские мобильные номера (форматы +7 и 8)
- Email — определяется по наличию символа @
- Серия и номер паспорта — 4 цифры серии и 6 цифр номера
- СНИЛС — формат XXX-XXX-XXX XX (11 цифр)
- ИНН — 10 или 12 цифр
- Полис ОМС — 16 цифр (различные форматы записи)

## Как запустить проект

### 1. Клонирование репозитория

git clone <URL_вашего_репозитория>
cd <название_папки_репозитория>

### 2. Установка зависимостей (бэкенд)

Убедитесь, что у вас установлен Python 3.10+.

pip install flask flask-cors opencv-python-headless numpy pytesseract torch transformers

Также необходимо установить Tesseract OCR:

- Скачайте установщик с официального репозитория Tesseract (https://github.com/UB-Mannheim/tesseract/wiki)
- Установите его. Путь к исполняемому файлу tesseract.exe должен быть указан в переменной окружения PATH, либо измените путь в файле server.py

### 3. Подготовка модели Gemma

Скачайте модель Gemma и поместите её в папку ./gemma/ в корне проекта. Сервер ожидает найти там токенизатор и веса модели.

### 4. Запуск бэкенда

python server.py

Сервер запустится на порту 8080.

### 5. Запуск фронтенда

Перейдите в папку с фронтендом и выполните команды для вашего фреймворка. Например:

npm install
npm run dev

## Как пользоваться сервисом

1. Откройте веб-интерфейс в браузере.
2. Выберите файл в формате JPEG или PNG, который нужно обработать.
3. Отметьте галочками те типы данных, которые необходимо заблюрить (ФИО, телефон, email, паспорт, СНИЛС, ИНН, ОМС).
4. Нажмите кнопку загрузки.
5. Через несколько секунд сервис выдаст обработанное изображение — его можно скачать. Все выбранные типы данных будут замазаны (заблюрены).
